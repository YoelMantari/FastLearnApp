🧠 Pregunta 61

¿Cuál es la característica distintiva del aprendizaje batch?

A. Entrenamiento incremental con nuevos datos
B. Entrenamiento único con todo el conjunto de datos disponible
C. Uso de recompensas y castigos
D. Procesamiento de datos sin etiquetas

✅ Correcta: B
🧾 Explicación: En aprendizaje batch, el modelo se entrena en un único paso usando todo el conjunto de datos.

🧠 Pregunta 62

¿Qué describe mejor el proceso de aprendizaje online?

A. Reentrenar el modelo desde cero con todos los datos cada vez que cambian
B. Ajustar el modelo de forma incremental con pequeños subconjuntos de datos
C. Ignorar los datos nuevos
D. Entrenar solo con datos etiquetados

✅ Correcta: B
🧾 Explicación: El aprendizaje online actualiza el modelo gradualmente a medida que llegan nuevos datos en mini-batches.

🧠 Pregunta 63

¿Cuál es una ventaja principal del aprendizaje batch?

A. Adaptación continua en tiempo real
B. Requiere pocos recursos iniciales
C. Resultados estables si los datos no cambian frecuentemente
D. Manejo sencillo de datos ruidosos

✅ Correcta: C
🧾 Explicación: El batch ofrece estabilidad en escenarios estáticos donde los datos no varían mucho.

🧠 Pregunta 64
